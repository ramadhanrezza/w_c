## How to run

+ clone this repository
+ create virtualenv inside the folder

    `virtualenv --python=python3 venv`
+ activate virtualenv

    `source venv/bin/activate`
+ install the requirements

    `pip install -r requirements.txt`
+ run crawler.py

    `python crawler.py`
+ run dbpush.py

    `python dbpush.py`
+ run server
    
    `python manage.py`
